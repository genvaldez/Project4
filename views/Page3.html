    Uses of Linear Algebra for Computer Science: Ranking in search engines, Linear Programming, Error correcting codes, Graphics, Facial recognition, and several others.

  • The most high profile use of linear algebra was in the creation og google.The orginal ranking algorithm has since become far more complicated, but it used linear algebra to rank what searches shold show up first.
  • The most widely used application of linear algebra is definitely optimization, and the most widely used kind of optimization is linear programming.
  • Another unseen but widespread use of linear algebra is in coding theory. The issue is to encode data in a way that if the encoded data is messed with, there is a possiblity to recver the uncoded data. This process is called error correcting codes.
  • Most graphic innovation since computers have existed has come from video games and movies. The important part of graphics is projecting a three dimentional scene onto a two dimentional screen. By using rotations and scaling.
